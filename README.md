## Overview
This project is a simple **Lightbox Image Viewer** built using **vanilla HTML, CSS, and JavaScript**.  
It allows users to click on image thumbnails and view a full-size preview in a lightbox overlay without navigating away from the page.

## Features
- Clickable image thumbnails
- Full-size image preview in a lightbox
- Overlay covers the entire viewport
- Close lightbox by clicking the close button or background
- Smooth and intuitive user experience
- No external libraries or frameworks

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## How It Works
- Thumbnails are displayed in a gallery layout
- Clicking a thumbnail opens the lightbox
- JavaScript dynamically updates the image source
- The lightbox is shown or hidden by toggling CSS display properties

## How to Run
1. Clone the repository
2. Open `index.html` in any modern browser
3. Click on any image thumbnail to open the lightbox
